Data Collection Strategy

We researched every platform's API. Here's what's actually available and our proposed ingestion approach.

Platform API Status Auth What We Get Our Plan
Ghost Full API JWT (Admin Key) Views, clicks, members, posts, newsletters API direct
LinkedIn Gated OAuth 2.0 + approval Post analytics, impressions (if approved) Playwright daily cron or XLS export
Substack No Official API N/A No official endpoints; unofficial libs fragile Playwright daily cron or RSS
Medium Deprecated N/A Nothing — API archived March 2023 Playwright daily cron or drop

Playwright MCP — Universal Fallback for Platforms Without APIs

For LinkedIn, Substack, and Medium — none of which offer usable analytics APIs — we propose a Playwright MCP server running as a daily cron job. It logs into each platform's analytics dashboard, scrapes the numbers, and writes structured data to Airtable. One request per platform per day.

Why this works at our scale

  • 1 request/platform/day — barely detectable
  • Near-zero cost (headless browser on a cron)
  • Universal — works for any platform with a web dashboard
  • No API approval or rate limits to worry about
  • Real browser fingerprint, not a headless bot signature

Risk: botting detection

  • Platforms can flag automated logins
  • LinkedIn is most aggressive at bot detection
  • Account suspension is possible if detected
  • At 1x/day frequency, risk is low but non-zero
  • Fallback: manual XLS/CSV export if flagged

Recommendation: Ghost via API (best path). LinkedIn, Substack, Medium via Playwright MCP daily cron. If LinkedIn API approval comes through, we migrate off Playwright. The API is always better long-term — Playwright is our practical bridge to get data flowing from day one.

Risk Register

Critical

Medium API is dead

API archived March 2023. No viable programmatic path to analytics data.

Fallback: RSS feed for post titles only, or drop Medium from scope entirely.

High

Substack has no official API

Unofficial Python libraries reverse-engineer endpoints. Could break at any time.

Fallback: RSS feed + manual data entry for key metrics.

High

LinkedIn API approval timeline

Analytics API requires OAuth + case-by-case approval. Could take weeks.

Fallback: Parse manual XLS export from LinkedIn analytics dashboard.

High

Playwright botting detection

LinkedIn is most aggressive at detecting automation. Substack and Medium less so.

Mitigation: 1 request/day/platform, real browser fingerprint, single account. Fallback: manual export.
